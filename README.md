# Описание проекта и инструкция по запуску

## Описание
Проект для изучения работы в FastAPI на примере задач, разбитых согласно матрице Эйзенхауэра

### Технологии
Python 3.13
fastapi 0.119
SQLAlchemy 2.0.45
alembic 1.17.2
pydantic 2.12.0
dotenv 0.9.9

### Запуск проекта
 1) Установить и активировать виртуальное окружение ```source venv/Scripts/activate```
 2) Установить зависимости из файла requirements.txt ```pip install -r requirements.txt```
 3) Запустить проект ```uvicorn main:app --reload```

### Внесенные изменения
- Теперь квадрант расчитывается относительно текущей даты и даты дедлайна
- Теперь можно получить статус задачи с указанием времени, оставшегося до выполнения
- Добавлено разделение по ролям и возможность регистрации и аутентификации
